# MusicGearSetup
Setup / backup files for music instrument

## Setup
Github limit file upload to 100mb.
Run this to increase the HTTP buffer issue: `git config http.postBuffer 524288000`

## Content
### TR8S
USB Driver for mac Sequoia : tr8s_mac14drv_m103.tgz

### Machinedrum
Various Sysex drop midi file for Machinedrum
+ SysExLibrarian to upload the file to the machinedrum via midi transfer

### MOTU
Audio driver for Motu 828x
USB driver for Motu Midi Express 128

### Minitaur
Preset from the net
Editor is too big, but can be found here:
https://www.moogmusic.com/downloads/?product=Minitaur&type=Editor

### Elektron
Overbridge software used for Digitakt
File is > 100MB, so it can be found here: 
https://www.elektron.se/support-downloads/overbridge
